# contentFirstBrowse（浏览模式内容优先）

`contentFirstBrowse` 是一个 NVDA 全局插件，用于优化浏览模式下的朗读顺序。

在浏览模式移动光标时，插件会优先朗读内容文本，再朗读角色/状态信息，减少“先报控件信息、后报正文”带来的干扰。

## 功能特点

- 浏览模式下启用“内容优先”朗读顺序。
- 安装后自动生效，无需额外配置。
- 仅影响符合条件的浏览模式光标移动，不改变其他场景行为。

## 适用条件

插件仅在以下条件同时满足时生效：

- 输出原因为 `CARET`（光标移动朗读）。
- 当前文本信息对象属于浏览模式。
- 当前不在焦点模式（`passThrough = False`）。
- 本次移动不是由焦点变化触发。

其余情况会回退到 NVDA 原始朗读逻辑。

## 兼容性

- 插件版本：`1.0.0`
- 最低 NVDA 版本：`2025.1`
- 已测试 NVDA 版本：`2025.3`

## 使用说明

- 本插件无快捷键、无设置面板。
- 安装并重启后自动生效。
- 在浏览模式中用方向键等阅读命令移动时可体验“内容优先”朗读。

## 已知限制

- 该插件通过补丁方式挂接 NVDA 内部语音函数实现。
- 如果 NVDA 后续版本修改了相关内部实现，可能需要更新插件以保持兼容。
